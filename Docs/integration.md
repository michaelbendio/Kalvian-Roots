Original Notes
--------------

The purpose of Kalvian Roots is to help the user to add source citations to FamilySearch. Source citations for a person are generated using data from Juuret Kälviällä. They will take the form of 
Information on page 244 includes:
Matti Erikinp., b. 14 March 1756
Kaarin Matint., b. 6 November 1760
m. 12 November 1779
Children:
→ Elias, b. 7 December 1781, m. Maria Iso-Passoja 27 May 1800
Maria, b. 8 May 1784
Brita Kaisa, b. 21 October 1794
Note:
Perhe muuttanut 1802 Kykyri II 7
Additional information:
Elias's marriage date is on page 244

Source citations for birth, death and marriage events are generated by queries of the hiski.genealogia.fi database. All of the children in a family can be gotten by querying the children born to a given father and given mother from a year before their marriage to 35 years after. Here is a hiski citation for the birth of Matti Herlevi: https://hiski.genealogia.fi/hiski?en+t4085059

I want to integrate Kalvian Roots with FamilySearch such that when I enter a family ID in Kalvian Roots I will see the parents and the union of children in Juuret Kälviällä and children returned by a hiski query and children in FamilySearch. Clicking on a name in a Juuret Kälviällä family will generate a source citation for that person as a child in a family. Clicking on a date will generate a hiski query result.

Both Juuret Kälviällä and hiski citations are copied to the system clipboard, ready for the user to paste in the appropriate place in FamilySearch.

Here is pseudo code for FamilySearch integration.

The person page shows the main person's birth and death dates and FamilySearch ID.
Add the Kalvian Roots source citation() {
   Navigate to the Sources tab by appending the FamilySearch ID to "www.familysearch.org/en/tree/person/sources" + FamilySearch ID
    If it doesn't include "Juuret Kälviällä : avuksi" {
    click "ADD SOURCE"
    click "Attach From Source Box"
    click "Juuret"
    click "ATTACH"
    add the Kalvian Roots source citation in the "Reason to Attach Source" text box
   else {
    done
   }
   Navigate to the person page ("https://www.familysearch.org/en/tree/person/details/" + FamilySearch ID)
}
Click on the spouse name
   get the birth and death years then click away
   Add the Kalvian Roots source citation()
for each of the children
   click on the name, get the birth and death years
   Add the Kalvian Roots source citation()
